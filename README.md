# 🚗 Car Cleaning Management System

A full-stack Car Cleaning Management System built using Vue 3, Node.js, Express, and MongoDB.

## Features

### Authentication

* Admin Login
* Worker Login
* JWT Authentication
* Role-Based Access Control

### Car Management

* Add New Cars
* Edit Car Details
* Deactivate Cars
* Search Cars

### Cleaning Management

* Mark Cars as Cleaned
* Daily Cleaning Tracking
* Dashboard Summary

### Payment Management

* Record Monthly Payments
* Track Paid & Pending Customers
* Monthly Revenue Calculation
* Payment History

### Dashboard

* Total Cars
* Monthly Revenue
* Paid Customers
* Pending Customers
* Daily Summary

### UI Features

* Responsive Mobile Design
* Modern Dashboard UI
* Sidebar Navigation
* Search & Filters

## Tech Stack

### Frontend

* Vue 3
* Vue Router
* Axios
* Bootstrap 5

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT
* bcrypt

## Installation

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
